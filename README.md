## 💻 Running Code Locally

To compile and run any C++ solution locally using `g++` or `clang++`:

```bash
# Navigate to specific day folder
cd Day01

# Compile with C++17 standard
g++ -std=c++17 hello_world.cpp -o hello_world

# Execute binary
./hello_world
```
